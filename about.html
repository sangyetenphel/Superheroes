About This Website

This website is dedicated to showcasing iconic Marvel heroes and villains, highlighting what makes each character unique. Marvel characters have inspired generations through stories of courage, responsibility, sacrifice, and growth, and this site celebrates those themes in a simple and engaging way.

I chose the Marvel theme because these characters are more than just superheroes — they represent different personalities, struggles, and values. From Iron Man’s intelligence and innovation to Spider-Man’s sense of responsibility and Hulk’s inner conflict, each character brings something meaningful to the Marvel universe.

The design of this website focuses on clarity and consistency. Using Bootstrap cards and a responsive layout helps keep the content organized and visually balanced across different screen sizes. The goal was to create a clean, easy-to-navigate site that allows users to explore characters without distraction.

Overall, this project combines creativity with technical skills, using HTML, CSS, and Bootstrap to build a themed website that is both informative and visually appealing.

This project also served as an opportunity to practice front-end development skills while working with a familiar and engaging theme. Building the site helped reinforce concepts such as layout structure, responsive design, and user-friendly navigation, while allowing room for creativity through visuals and content organization.
